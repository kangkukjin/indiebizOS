# IndieBiz OS 상세 설명서

IndieBiz OS는 개인과 소규모 비즈니스를 위한 AI 기반 통합 운영체제입니다. 단순한 챗봇을 넘어 사용자의 인격을 반영한 에이전트 팀을 구성하고, 복잡한 업무를 자동화하며, 탈중앙화된 네트워크를 통해 소통할 수 있는 환경을 제공합니다.

## 1. 핵심 가치 및 차별점

### 나만의 AI 인격(페르소나) 설계
단순한 역할 대행이 아니라, 에이전트의 말투, 전문성, 대화 스타일을 직접 정의할 수 있습니다. 각 에이전트는 사용자의 맥락(복용 약, 선호도, 과거 대화 등)을 기억하는 개인화된 전문가로 동작합니다.

### 원클릭 자동화 (스위치)
반복되는 AI 작업을 '스위치'로 저장하여 한 번의 클릭으로 실행할 수 있습니다. 뉴스 수집, 요약, 리포트 생성 등 복잡한 워크플로우를 자연어로 정의하고 예약 실행(Cron)할 수 있습니다.

### 데이터 주권 및 프라이버시
모든 데이터는 사용자의 로컬 컴퓨터에 저장됩니다. Ollama를 통한 로컬 LLM 지원으로 오프라인 환경에서도 작동 가능하며, 플랫폼 종속성 없이 자유롭게 AI 모델을 선택할 수 있습니다.

### P2P 네트워크 (IndieNet)
Nostr 프로토콜을 기반으로 한 탈중앙화 네트워크를 지원합니다. 중앙 서버 없이 암호화된 DM을 주고받거나 비즈니스 파트너와 안전하게 연결될 수 있습니다.

## 2. 주요 기능 및 시스템 구성

### 프로젝트 및 에이전트 팀
- **독립된 작업 공간**: 프로젝트별로 대화 기록, 에이전트 설정, 컨텍스트가 완전히 격리됩니다.
- **위임 체인 (Delegation Chain)**: 에이전트 간에 작업을 위임하고 결과를 자동으로 보고받는 협업 시스템을 갖추고 있습니다.
- **다중채팅방**: 여러 프로젝트의 에이전트를 한자리에 소환하여 그룹 대화를 나눌 수 있습니다.

### 도구 패키지 시스템
현재 20개의 전문 도구 패키지가 설치되어 있으며, 에이전트는 필요한 도구만 동적으로 로드하여 사용합니다.
- **주요 도구**: 웹 검색(Web), 데이터 시각화(Visualization), 미디어 제작(Media Producer), 금융 분석(Investment), 시스템 관리(PC Manager) 등.

### 비즈니스 자동응답 V3
외부 채널(Gmail, Nostr)로 들어오는 메시지를 AI가 판단하여 비즈니스 DB 검색 후 즉시 응답을 생성하고 발송하는 지능형 시스템을 제공합니다.

## 3. 기술 아키텍처
- **Back-end**: Python FastAPI 기반의 통합 AI 코어 (AIAgent 클래스). 스트리밍 응답 및 XML 태그 구조의 정교한 프롬프트 제어.
- **Front-end**: Electron + React (TypeScript) 기반의 데스크톱 애플리케이션.
- **AI Providers**: Claude, GPT, Gemini, Ollama 등 다양한 모델 지원.
