# IndieBiz OS: 개인과 소규모 비즈니스를 위한 AI 운영체제

## 1. 개요
IndieBiz OS는 단순한 AI 챗봇을 넘어, 사용자가 직접 AI 인격(Persona)을 설계하고 업무를 자동화하며 탈중앙 네트워크로 연결되는 완전한 개인용 AI 운영체제입니다. "하나의 만능 AI보다 여러 목적별 AI 연합이 낫다"는 철학 아래, 프로젝트 단위의 격리된 작업 공간과 전문 에이전트 팀을 제공합니다.

## 2. 핵심 가치 및 차별점
- **AI 페르소나 설계**: 단순한 역할 부여를 넘어 대화 스타일, 전문 지식, 기억을 가진 고유한 인격을 정의합니다.
- **원클릭 자동화 (Switch)**: 반복적인 프롬프트 작업을 '스위치'로 저장하여 한 번의 클릭이나 스케줄링으로 실행합니다.
- **데이터 주권**: 모든 데이터는 사용자의 로컬 환경에 저장되며, Ollama를 통한 로컬 LLM 지원으로 완전한 오프라인 작동 및 보안을 보장합니다.
- **탈중앙 네트워크 (IndieNet)**: Nostr 프로토콜을 사용하여 중앙 서버 없이 전 세계 사용자 및 비즈니스 파트너와 안전하게 소통합니다.

## 3. 기술적 아키텍처
### 3.1 통합 AI 코어 (AIAgent)
시스템 AI와 프로젝트 에이전트는 동일한 `AIAgent` 클래스를 공유합니다. 이는 프로바이더(Claude, GPT, Gemini, Ollama) 통합, 스트리밍 처리, 도구 실행 로직을 일원화하여 시스템 전반에 걸쳐 일관된 성능과 확장성을 제공합니다.

### 3.2 위임 체인 시스템 (Delegation Chain)
에이전트 간의 협업을 위해 단일/순차/병렬 위임을 지원합니다. `call_agent` 도구를 통해 작업을 분배하고, 하위 에이전트의 작업이 완료되면 자동으로 결과를 취합하여 상위 에이전트에게 보고하는 비동기 협업 구조를 가집니다.

### 3.3 XML 기반 프롬프트 구조
AI의 파싱 정확도를 극대화하기 위해 모든 프롬프트와 메시지 히스토리에 XML 태그 구조를 적용합니다. 이는 복잡한 지시사항과 컨텍스트를 AI가 오차 없이 이해하도록 돕습니다.

## 4. 주요 기능 및 모듈
- **프로젝트 시스템**: 목적별로 격리된 무제한 작업 공간.
- **도구 패키지 시스템**: 21개의 기본 도구(Web, Python, Investment, Media 등)를 에이전트별로 동적 로딩.
- **자동응답 V3**: Tool Use 기반의 단일 AI 호출로 외부 메시지(Gmail, Nostr)에 대한 판단, 검색, 응답 발송을 통합 처리.
- **스케줄러**: Cron 표현식을 이용한 자동 작업 실행 및 리포트 생성.
- **다중채팅방**: 여러 프로젝트의 에이전트를 한곳에 소환하여 협업하는 그룹 채팅 환경.

## 5. 비즈니스 확장성
IndieBiz OS는 이웃(Neighbors) 관리 시스템과 통신 채널 통합을 통해 비즈니스 운영을 지원합니다. 사용자의 근무 지침과 비즈니스 문서를 학습한 AI가 외부 문의에 대해 실시간으로 대응하며, 모든 과정은 사용자의 감독 하에 이루어집니다.
