# IndieBiz OS: 개인과 소규모 비즈니스를 위한 AI 운영체제

IndieBiz OS는 단순한 챗봇을 넘어, 사용자가 직접 AI 인격을 설계하고 복잡한 업무를 자동화하며 탈중앙화 네트워크로 연결되는 혁신적인 개인용 AI 운영체제입니다.

## 1. 핵심 가치 및 철학
IndieBiz OS는 '하나의 만능 AI보다 여러 목적별 AI가 낫다'는 철학을 바탕으로 합니다. 사용자는 각기 다른 페르소나와 전문 지식을 가진 에이전트 팀을 구성하여 맥락이 격리된 효율적인 작업 환경을 구축할 수 있습니다. 모든 데이터는 사용자의 로컬 환경에 저장되어 데이터 주권을 완벽히 보장합니다.

## 2. 주요 기술적 특징
### 2.1. 통합 AI 아키텍처
시스템 AI와 프로젝트 에이전트가 동일한 `AIAgent` 코어 클래스를 공유합니다. 이를 통해 Anthropic Claude, OpenAI GPT, Google Gemini, Ollama(로컬 LLM) 등 다양한 프로바이더를 일관되게 지원하며, 실시간 스트리밍과 도구 실행 로직이 일원화되어 있습니다.

### 2.2. 에이전트 위임 체인 (Delegation Chain)
에이전트 간 비동기 협업을 위한 핵심 메커니즘입니다. `call_agent` 도구를 통해 단일, 순차, 병렬 위임이 가능하며, 작업 완료 시 자동으로 결과가 보고되는 체계적인 협업 구조를 가집니다. 시스템 AI 역시 프로젝트 에이전트에게 작업을 위임할 수 있는 '시스템 AI 위임' 기능을 갖추고 있습니다.

### 2.3. 동적 도구 패키지 시스템
에이전트는 실행 시 필요한 도구만 동적으로 로딩합니다. 현재 21개의 도구 패키지(Android 관리, 금융 데이터 분석, 통계청 데이터 조회, 미디어 제작 등)가 설치되어 있으며, 사용자는 직접 패키지를 개발하거나 Nostr 네트워크를 통해 공유 및 설치할 수 있습니다.

### 2.4. 비즈니스 자동응답 V3
Tool Use 기반의 단일 AI 호출 방식을 통해 외부 메시지(Gmail, Nostr)를 분석하고, 비즈니스 DB 검색 및 응답 발송을 한 번에 처리합니다. 이는 기존의 2단계 처리 방식을 혁신적으로 개선한 결과입니다.

## 3. 주요 기능 구성
- **프로젝트 시스템**: 독립적인 작업 공간, 대화 기록, 에이전트 설정을 가집니다.
- **스위치 & 스케줄러**: 반복적인 프롬프트를 자동화로 저장하고 cron 표현식으로 예약 실행합니다.
- **IndieNet (P2P)**: Nostr 프로토콜을 활용한 탈중앙화 통신 및 비즈니스 네트워크를 지원합니다.
- **다중채팅방**: 여러 프로젝트의 에이전트를 한곳에 소환하여 그룹 대화를 수행합니다.

## 4. 시스템 구조
- **Backend**: Python FastAPI 기반으로 AI 코어, 스케줄러, 채널 폴러 등을 운영합니다.
- **Frontend**: Electron + React (TypeScript) 기반의 데스크톱 애플리케이션입니다.
- **Data**: SQLite DB(비즈니스, 대화 기록 등)와 YAML/JSON 설정 파일을 사용합니다.

IndieBiz OS는 사용자가 자신의 비즈니스와 일상을 AI와 함께 설계하고 자동화할 수 있는 강력한 기반을 제공합니다.
