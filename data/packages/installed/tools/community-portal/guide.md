# 개인 포털 (community-portal) 가이드

"커뮤니티당 노드 하나" 전략의 홈. 가입을 받는 공개 홈(`/h/<5자 코드>/`)으로 이웃을 모으고,
이웃의 **레벨**에 따라 콘텐츠(가족신문 `/n/`·공개파일 `/s/`)와 **계기**(🎨 아이콘, 🎵 유튜브뮤직 등)를
indiebizOS 설치 없이 브라우저로 빌려 쓰게 한다.

## 구조 (3층)

| 층 | 파일 | 역할 |
|---|---|---|
| 운영자 어휘 | `handler.py` `[others:portal]` | 이웃 승급·개인 링크 발급/회수·진열 다이얼·감사로그 |
| 공개 서빙 | `backend/api_portal.py` | 홈 렌더·가입·개인 링크 쿠키·**회원 실행 게이트** |
| 공개 게이트 | public-files Worker `/h/` | 무캐시 프록시(개인화 HTML — 캐시 절대 금지), Cookie 왕복 |

상태(`data/portal_state.json`)·게이트 로직은 `portal_core.py` 단일 소스 — 운영자 어휘와
공개 서빙이 공유(flock 직렬화). ★portal_core 수정 시 백엔드 재시작 필요(api_portal 이 캐시).

## 핵심 규칙

- **포털은 여러 개** — 대상별 주소(가족용·친구용, showcase 바스켓 패턴). 각 포털이 자기 회원
  명부·진열(첫페이지 구성)을 독립으로 가진다. 새 포털=진열 전부 꺼진 채 시작(안전 기본).
- **가입/로그인 = 아이디+비밀번호(네이버식)** → 레벨 0 자동 등록+즉시 로그인(승급이 곧 승인).
  자동 로그인(기본)=1년 쿠키. pbkdf2 해시 저장. 로그인 무차별 대입=IP당 분당 10회 제한.
- **개인 링크 = 운영자 발급 즉시 로그인 열쇠** (`/h/<슬러그>/k/<회원키>`) — **비밀번호 분실
  복구 경로**(재설정 메일 불필요). 유출·이탈=revoke(그 사람의 링크·쿠키·로그인 전부 차단).
  홈 주소 자체는 완전 공개 가능.
- **다이얼**: 항목마다 `{enabled, min_level, guest_daily, member_daily, global_daily}`.
  기본값은 노드 축으로 유도 — engines(비용 발생)=회원+한도 / sense(조회)=회원 1+(운영자가
  손님 0으로 내릴 수 있음) / **self·limbs·others 가 낀 계기는 목록 자체에 없음**(다이얼 불가).
  예외는 `portal_core.MANUAL_LISTABLE` 에만 명시(현재 ytmusic — ▶이 mode:"client" 리터럴 강제).
- **회원 실행 게이트**: 범용 /ibl/execute 직결 금지. 계기 `app:` 블록에 선언된 액션 템플릿의
  *인스턴스*만 실행 — 리터럴 파라미터(op·mode 등)는 일치 강제, `$key`·`{field}` 자리만 값 허용.
  - 한도 3중: 회원별 + 손님 IP별 + 계기 전역 일일 캡. 기록은 `data/portal_audit.jsonl`.
- 크롤형 계기(직방·당근·여기어때 등)는 남용 시 집 IP가 차단됨(팔의 평판) — 손님에게 열지 말 것.

## 첫 사용 흐름 (운영자)

1. 포털 탭에서 대상별 주소 생성 (`{op: "create", title: "우리 가족"}`).
2. 진열 탭에서 그 포털 첫페이지에 보여줄 것을 ▼열기 (`{op: "display", portal: …, key: …, level_down: true}`).
3. 주소를 공유 → 상대가 홈에서 아이디·비밀번호로 가입(레벨 0 자동).
4. 이웃 탭에서 ▲승급 → 회원 계기 열림. (직접 등록은 `{op: "join", name: …}` = 링크 전용 회원.)
5. 비밀번호 분실 신고가 오면 `{op: "issue", member_id: …}` 새 링크 전달(즉시 로그인).

## 함정

- 아이콘 계기: 브라우저에선 클립보드 자동복사 불가(그건 폰 네이티브 Kotlin 경로) —
  이미지를 "꾹 눌러 복사" 안내. 비용 ~93원/장(gemini) → member_daily 기본 20.
- 유튜브뮤직: googlevideo URL 은 맥 IP 에 잠김 → 게이트가 stream_url 을 **오디오 프록시**
  (`/h/<슬러그>/tune/<video_id>`)로 바꿔쳐 외부망(LTE)에서도 재생됨. 맥이 집 IP 로 받아
  Range 통과로 중계, 만료 시 1회 재해소, 수제 URL(캐시 미스)은 사용량 카운트. 회원 쿠키
  필수(익명이면 유튜브 공개 프록시가 되므로 절대 안 엶). '큐' 탭(맥 스피커 제어)은
  phone_render:false 라 포털에 안 나감.
- Worker 에서 `/h/` HTML 캐시 절대 금지(쿠키 개인화). `no-store`.
