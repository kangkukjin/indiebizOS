# Information

에이전트가 다양한 외부 API를 통해 정보를 수집하고 콘텐츠를 제작할 수 있게 합니다.
명언, 날씨, 여행 정보, 뉴스 등 폭넓은 데이터에 접근합니다.

## 주요 기능
- API Ninjas: 명언, 역사, 날씨, 사전 등 다양한 지식 정보
- 공개 API 검색: 전 세계 공개 API 라이브러리 검색
- 여행 정보: 항공권, 호텔, 관광지 실시간 조회 (Amadeus)
- 콘텐츠 제작: HTML 신문, 잡지 형식 보고서 자동 생성

## 적합한 에이전트
- 정보 수집/리서치 에이전트
- 콘텐츠 제작 에이전트
- 여행 관련 에이전트
