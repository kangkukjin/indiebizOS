# PC Manager - AI 파일 탐색기

## 개요
GUI 기반 파일 탐색 도구. AI 에이전트가 창을 열어주고, 사용자가 시각적으로 파일을 탐색할 수 있다.

## 도구

### `open_file_explorer`
PC Manager GUI 창을 연다.
- 파라미터: `path` (선택) - 초기 경로
- 동작: Electron 윈도우 실행

**사용 예시:**
```
에이전트에게: "내 문서 폴더 열어줘"
에이전트에게: "외장하드 확인해줘"
에이전트에게: "PC 관리창 열어"
```

## 현재 기능

### 파일 탐색
- 사이드바: 홈, 드라이브 목록 (내장/외장)
- 파일 목록: 폴더/파일 표시, 크기 표시
- 네비게이션: 상위 폴더, 홈, 브레드크럼

## 계획 (추후 추가)

### AI 분석
- 폴더별 요약 (파일 종류, 용량 분포)
- 중복 파일 하이라이트
- 정리 제안 ("6개월 미접근 폴더")

### 정리 액션
- 파일 이동/삭제 (휴지통)
- 카테고리별 자동 분류
- 용량 시각화 (트리맵)
